# Jodahen.nl – Personal Games Website

This project is a **browser-based personal games website** that hosts multiple interactive games including Wordle and other mini-games. The site is designed to showcase game development skills using **HTML, CSS, and JavaScript** while providing fun, interactive experiences.

The goal is to demonstrate interactive browser-based games in a **portfolio-friendly environment** and provide reusable code for learning purposes.

---

## What This Project Includes

### Games

* **Wordle** (5 & 6 letter versions, random words optional)
* **Reaction Test** (F1 styled reaction test)
**Barcode Generator** (Generate a barcode of your chosen tekst!)
**QR Code Generator** (Generate a QR of your chosen tekst (or link))
**Morse Code Translator / Generator** (Translate a morse code string, or Generate on with your own message)
* Mini-games demonstrating JavaScript logic and UI interaction

## Technologies Used

* **HTML5** – Structure of the website and games
* **CSS3** – Styling, responsive design, and animations
* **JavaScript** – Game logic, interactivity, and dynamic content
* **Git & GitHub Pages** – Version control and optional hosting

---

## Running the Website

### To run locally:

1. Download or clone the repository.
2. Open `index.html` in a browser.

> ⚠️ Note: Some games require word lists (`.txt` files). For live deployment, word lists should be hosted on the server or fetched via HTTPS to avoid CORS issues.
